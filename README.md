# Typing-fast
Лабораторная по репрезиториям: Приложение для тренировки скорости печати
Создатель: Трайбер Валерия Владимировна  
Группа: ИСП(9)-23-1  
Дата начала: [1.03.2026]

## Описание
Разработка приложения для тренировки скорости печати в две итерации:
1. Базовая версия (фиксированный текст, подсчет символов в минуту)
2. Версия с настройкой уровня сложности (легкий, средний, сложный)
